Цель практики
Закрепление теоретических знаний и приобретение практических навыков в области клиентской веб-разработки, а также формирование умений проектировать, реализовывать и размещать веб-приложения с использованием современных технологий (HTML5, CSS3, JavaScript ES6+) и систем контроля версий (Git, GitHub).
Задачи практики
1.	Изучить современные стандарты вёрстки (HTML5) и стилизации (CSS3) веб-интерфейсов.
2.	Освоить работу с DOM-моделью документа и обработку событий на стороне клиента средствами JavaScript.
3.	Разработать пользовательский интерфейс веб-приложения согласно техническому заданию.
4.	Реализовать алгоритмы обработки и валидации пользовательского ввода (парсинг даты, расчёт разности дат, определение високосного года).
5.	Провести тестирование разработанного приложения на корректность работы и устойчивость к некорректным данным.
6.	Оформить проект в виде репозитория и разместить его на платформе GitHub.

Объект исследования
Процесс разработки клиентской части веб-приложения с использованием технологий HTML, CSS и JavaScript.

Предмет исследования
Методы и алгоритмы обработки дат в JavaScript, приёмы валидации пользовательского ввода, а также способы взаимодействия с DOM-деревом для динамического отображения результатов вычислений.
Формулировка задачи
Создать пользовательский интерфейс, состоящий из поля ввода (<input>), кнопки (<button>) и двух блоков (<div>), расположенных по центру экрана браузера.
Реализовать функцию, принимающую строку в формате дд.мм.гггг и вычисляющую количество дней от введённой даты до 31 декабря того же года.
По нажатию кнопки вывести в первый блок количество оставшихся дней до Нового года.
Реализовать функцию определения високосности года по правилу: год является високосным, если он делится нацело на 4, не кратен 100, либо кратен 400.
По нажатию кнопки вывести во второй блок информацию о том, является ли год введённой даты високосным или нет.
Средства разработки
Языки разметки и стилей: HTML5, CSS3
Язык программирования: JavaScript 
Среда разработки: Visual Studio Code
В ходе прохождения практики были выполнены следующие работы:
Анализ требований и проектирование — изучено техническое задание, определены входные и выходные данные, спроектирована структура интерфейса и логика работы приложения.
Разработка пользовательского интерфейса — создана HTML-разметка с полем ввода, кнопкой и двумя блоками результатов; разработаны CSS-стили для центрирования элементов и оформления состояний (фокус, hover, ошибки).
Программная реализация логики — написаны функции на JavaScript для парсинга даты в формате дд.мм.гггг с валидацией, расчёта количества дней до 31 декабря и определения года по правилам григорианского календаря.
Интеграция компонентов — подключены обработчики событий кнопки и клавиши Enter, настроен динамический вывод результатов и сообщений об ошибках в соответствующие блоки страницы.
Тестирование — проведена проверка корректности работы на валидных данных, граничных случаях (високосные годы, конец года) и некорректном вводе; выполнена кросс-браузерная проверка.
Оформление и размещение проекта — код размещён в репозитории на GitHub, создан README.md с описанием и инструкцией, настроена публикация через GitHub.
